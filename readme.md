# 🌿 FarmSphere - Smart Grow Monitor Dashboard

FarmSphere is a high-fidelity, visually rich web dashboard interface designed for modern smart farming, hydro-culture monitoring, and autonomous agriculture tracking. Inspired by professional illustrative design blueprints, the application combines a dark glassmorphic styling system with real-time environment animations—including continuous drifting skies, ambient mist layers, and living vector butterfly simulations.

---

## ✨ Interface Design Highlights

* **Vibrant Glassmorphic HUD:** Dark, translucent glass cards equipped with responsive layout glows corresponding directly to the health matrix rules (pH Balance, Water Temperature, Humidity, and Nutrient Volume).
* **Dynamic Background Weather:** Multi-tiered procedural CSS clouds drifting across the skyline at asymmetrical speeds to replicate organic nature cycles.
* **Vector Flight Swarm Simulation:** Low-overhead JavaScript engine animating pathing nodes and mathematical rotations for bioluminescent butterflies cross-pollinating the frame.
* **Greenhouse Rack Visualizer:** An automated hydro-shelf graphic featuring built-in 3D plant bobs and high-intensity simulated grow-light radiation masks.
* **Interactive Viewport Switching:** A dynamic tab routing controller built entirely in vanilla JavaScript that swaps active layouts instantly without page reloads.
